# data/

Öll gögn hópsins eiga heima hér, skipt í tvennt: `raw/` og `processed/`.

**Git-siðareglur fyrir gögn:**

- Ekki vista viðkvæm gögn (lykilorð, persónuupplýsingar) — notið `.env` og `.gitignore`
- Ef gagnaskrár eru stórar (>50 MB), íhugið [Git LFS](https://git-lfs.com/) eða `.gitignore`
  þær og skjalið hvernig á að sækja þær í staðinn
- Nefnið skrár lýsandi og með dagsetningu ef við á, t.d. `atkvædi_2026-09-01.csv`
