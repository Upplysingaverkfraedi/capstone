# src/

Allur keyranlegur kóði hópsins — SQL, Python, R, eða annað.

**Nafnavenja:** Notið tölusetningu til að sýna keyrsluröð, t.d.:

```
01_sotta_gogn.py
02_hreinsa_regex.py
03_sql_greining.sql
04_myndraen_framsetning.R
```

**Siðareglur:**

- Ein skrá = eitt skref í vinnuflæðinu, ekki allt í einni risaskrá
- Skjalið að ofan í hverri skrá hvað hún gerir og hvað hún þarf til að keyra
- Engin hörð kóðuð lykilorð/API-lyklar — notið umhverfisbreytur (sjá `.env.example`)
